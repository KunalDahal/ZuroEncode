# Encode Bot

A Telegram bot for encoding media files with customizable settings.

## Features

- Encode the media in different Resolution
- Encode media with customizable settings like custom CRF, Preset, Audio Bit rate, CODEC, etc.
- Upload encoded files back to Telegram 
- Auto rename with encoding feature available
- Task queue management, 1 task a time
- User-specific settings
- FFMPEG encoding
- Using Pyrogram session for Services.
